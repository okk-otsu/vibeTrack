# vibeTrack

**vibeTrack** — минималистичное приложение для отслеживания времени учёбы и концентрации.
Помогает фиксировать учебные сессии, анализировать прогресс и визуально видеть, куда уходит время — по дням, неделям.

Проект вдохновлён интерфейсом iOS Screen Time и аналогичными приложениями вроде YPT.

## Обзор
### Возможности
- Создание дисциплин (предметов / активностей)
- Запуск, остановка, создание, редактирование времени учебных сессий
- Только одна активная сессия одновременно
- Статистика за неделю
  

### Интерфейс
- Минималистичный дизайн
- Поддержка Light / Dark Mode
- Богатый выбор цвета для каждой дисциплины
- Анимации и плавные переходы
- Адаптация под iPhone и macOS


### Платформы
- [x] iOS (17+)
- [x] macOS (14+)


## Запуск проекта
  1.	Клонировать репозиторий:
  ```git clone https://github.com/okk-otsu/vibeTrack.git```
  2.	Открыть vibeTrack.xcodeproj
  3.	Выбрать iPhone или My Mac
  4.	Run ▶︎

## Установка приложения на Mac
Обязательное условие: скачать Xcode
1.	Клонировать репозиторий: ```git clone https://github.com/okk-otsu/vibeTrack.git```
2.	Открыть vibeTrack.xcodeproj
3.	Выбрать My Mac
4.	Run ▶︎
5.	Запущенное приложение появится в **Dock**
6.	Нажми пкм по иконке приложения в **Dock**
7.	Параметры
8.	Показать в **Finder**
9.	Копируете запущенную программу в папку **программы**
